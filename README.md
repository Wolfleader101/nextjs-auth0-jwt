> A thin utility library for nextjs-auth0 to support JWT bearer tokens for API calls

## Install

```bash
npm install nextjs-auth0-jwt
```

## Usage

```ts
import { myPackage } from "my-package-name";

myPackage("hello");
//=> 'hello from my package'
```
